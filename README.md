## &lt;cosmoz-autocomplete&gt;

**cosmoz-autocomplete** is a haunted auto-complete component.

## Getting started

### Installing

Using npm:
```bash
npm install --save @neovici/cosmoz-autocomplete
```

### Usage

``` javascript
html`<cosmoz-autocomplete .source=${["A", "B","C", "D"]} />`;
```
